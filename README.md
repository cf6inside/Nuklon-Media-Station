🌌 Nuklon Media Station
Nuklon Media Station is a modern, fast, and user-friendly desktop application that allows you to download online media in high quality and convert the formats of your existing media files.

The project is built entirely on Python using Object-Oriented Programming (OOP) architecture, powered by the yt-dlp and FFmpeg infrastructures.

✨ Key Features
Modern and Sleek Interface: A fluid user experience designed with CustomTkinter, featuring dark/light theme support.

Advanced Download Engine: Powered by yt-dlp, offering playlist support and the ability to download in various resolutions (1080p, 720p, MP3 only, etc.).

Batch Format Converter: Integrated with FFmpeg and FFprobe to select multiple files simultaneously and convert them to formats like MP4, MP3, MKV, GIF, and WAV, accompanied by a real-time progress bar.

Smart Infrastructure Control: Automatically checks the yt-dlp version upon startup and directs the user to the update if it is outdated.

Dual Language Support: Instantly switchable 🇹🇷 Turkish and 🇬🇧 English interface with a single click.

🛠️ Technologies Used
Language: Python 3.x

Interface (GUI): CustomTkinter, Tkinter, Pillow

Backend Engines: yt-dlp (Downloading), FFmpeg & FFprobe (Conversion and Duration Measurement)

Packaging: PyInstaller

🚀 Installation and Usage
For End-Users (Portable Usage)
If you wish to use the application directly without installing any Python modules:

Navigate to the Releases tab on the right.

Download the latest Nuklon_Medya_Merkezi.exe file.

Double-click the downloaded file to start using the application immediately.

For Developers (Running from Source Code)
If you wish to compile or develop the project on your own machine:

1. Clone the repository:
   git clone https://github.com/cf6inside/Nuklon-Media-Station.git
2. Install the required libraries:
   pip install customtkinter yt-dlp Pillow pyinstaller
3. Ensure that the ffmpeg.exe and ffprobe.exe files are located within the project directory.
4. Launch the application
   python indirici.py

Nüklon Medya İstasyonu, internet üzerindeki medyaları yüksek kalitede bilgisayarınıza indirmenizi ve elinizdeki medya dosyalarının formatlarını değiştirmenizi sağlayan modern, hızlı ve kullanıcı dostu bir masaüstü uygulamasıdır. 

Proje, tamamen **Nesne Yönelimli Programlama (OOP)** mimarisiyle Python üzerinde inşa edilmiş olup, gücünü `yt-dlp` ve `FFmpeg` altyapılarından almaktadır.

## ✨ Öne Çıkan Özellikler

* **Modern ve Şık Arayüz:** CustomTkinter ile tasarlanmış, karanlık/aydınlık tema destekli akıcı kullanıcı deneyimi.
* **Gelişmiş İndirme Motoru:** `yt-dlp` altyapısı sayesinde oynatma listeleri (playlist) desteği ve istenilen çözünürlükte (1080p, 720p, Sadece MP3 vb.) indirme imkanı.
* **Toplu Format Dönüştürücü (Batch Processing):** `FFmpeg` ve `FFprobe` entegrasyonu ile birden fazla dosyayı aynı anda seçip saniyelik okuma yapan ilerleme çubuğu eşliğinde mp4, mp3, mkv, gif, wav gibi formatlara çevirme.
* **Akıllı Altyapı Kontrolü:** Uygulama açılışında yt-dlp sürümünü kontrol eder ve eskiyse kullanıcıyı yeni sürüme yönlendirir.
* **Çift Dil Desteği:** Tek tıkla anında değişebilen 🇹🇷 Türkçe ve 🇬🇧 İngilizce (English) dil arayüzü.
